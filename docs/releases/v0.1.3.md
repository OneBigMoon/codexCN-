# Codex CN++ v0.1.3

本版修复 Codex 插件市场/推荐区再次显示英文的问题。

## 下载哪个文件

- `CodexCNPlusPlus-macos.zip`：推荐普通用户下载。解压后得到 `Codex CN++.app`，可拖入 `/Applications`。
- `CodexCNPlusPlus-macos-0.1.3.dmg`：DMG 安装包，适合习惯拖拽安装的用户。
- Source code：适合开发者继续维护词典、补丁规则和打包脚本。

## 本版修复

- 新增扫描 `~/.codex/cache/codex_app_directory/*.json`，覆盖 Codex 下载的插件市场目录缓存。
- 对 app directory 大索引只做精确词典替换，避免把未知第三方 App 描述泛化成模板文案。
- 补充前端 `app.asar` 中的推荐区硬编码文案：`Prepare sales work faster`、`Communication`、`Developer Tools`。
- 新增测试复现 app directory 缓存漏扫问题。

## 根因说明

Codex 不只从 `plugins/cache` 和 `vendor_imports` 读取插件/技能文案，还会从 `~/.codex/cache/codex_app_directory` 读取一份下载的市场目录索引。旧版工具没有扫描这个目录，所以 Codex 刷新该索引后，界面会再次显示英文。

## 本地验收

本版发布前确认：

- `npm test` 全部通过。
- `npm run dry-run` 在已汉化状态下返回 `changedFiles: []`。
- `app.asar`、`codex_app_directory`、`skills-curated-cache.json` 中，本次用户反馈的关键英文命中为 `[]`。
