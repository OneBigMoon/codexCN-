# Codex CN++ v0.1.1

本版修复 Codex 更新后插件和技能列表重新出现英文分类、标题和描述的问题。

## 下载哪个文件

- `CodexCNPlusPlus-macos.zip`：推荐普通用户下载。解压后得到 `Codex CN++.app`，可拖入 `/Applications`。
- `CodexCNPlusPlus-macos-0.1.1.dmg`：DMG 安装包，适合习惯拖拽安装的用户。
- Source code：适合开发者继续维护词典、补丁规则和打包脚本。

## 本版修复

- 补充 Codex 新版插件市场分类汉化：`Business & Operations`、`Communication`、`Creativity`、`Data & Analytics`、`Education & Research`、`Finance`、`Other` 等。
- 分类汉化加入兜底规则，不只依赖单个固定文件中的精确文案。
- 新增测试覆盖新版插件市场分类，降低 Codex 下次更新后再次遗漏的概率。
- 已重新应用到本机 Codex 缓存、插件缓存和 `vendor_imports`。

## 已知说明

- 本工具只处理 Codex 插件和技能列表相关文案，不追求汉化 Codex 全部聊天或系统界面。
- 如果应用后界面仍显示旧英文，请完全退出并重新打开 Codex。
- Codex 更新后可再次点击 `应用汉化`，工具会重新扫描版本化缓存目录。

## 本地验收

本版发布前确认：

- `npm test` 全部通过。
- `npm run dry-run` 在已汉化状态下返回 `changedFiles: []`。
- 针对本次遗漏的英文分类和描述，关键缓存文件扫描结果为 `totalHits: 0`。
