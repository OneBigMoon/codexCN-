# Codex CN++ v0.1.0

首个可用版本，面向 macOS 本地 Codex 插件和技能列表汉化。

## 下载哪个文件

- `CodexCNPlusPlus-macos.zip`：推荐普通用户下载。解压后得到 `Codex CN++.app`，可拖入 `/Applications`。
- `CodexCNPlusPlus-macos-0.1.0.dmg`：DMG 安装包，适合习惯拖拽安装的用户。
- Source code：适合开发者继续维护词典、补丁规则和打包脚本。

## 本版包含

- macOS 菜单栏 App，菜单栏显示 `CN++`。
- 菜单项：`应用汉化`、`恢复最近备份`、`刷新状态`、`打开网页界面`、`打开日志`、`退出`。
- 原创 Logo 和 macOS App 图标。
- 内置 Node 运行时，App 可直接运行本地补丁工具。
- 支持扫描、预演、应用汉化、恢复最近备份。
- 覆盖 Codex 技能和插件列表显示文案，包括本地缓存、插件缓存、技能目录、`vendor_imports` 和部分前端目录标题。
- 每次应用前自动创建备份批次。
- Codex 更新后可重复点击 `应用汉化` 重新补丁。

## 已知说明

- 本工具只处理 Codex 插件和技能列表相关文案，不追求汉化 Codex 全部聊天或系统界面。
- 如果应用后界面仍显示旧英文，请完全退出并重新打开 Codex。
- 品牌名通常保留原文，例如 `GitHub`、`Slack`、`NVIDIA`、`Canva`。
- 如果 macOS 提示无法验证开发者，请右键点击 `Codex CN++.app`，选择 `打开`。

## 维护者打包

```bash
npm run build:mac-app
npm run dist:dmg
```

产物：

```text
dist/Codex CN++.app
dist/CodexCNPlusPlus-macos.zip
dist/CodexCNPlusPlus-macos-0.1.0.dmg
```

## 本地验收

本版发布前应确认：

- `npm test` 全部通过。
- `npm run dry-run` 在已汉化状态下返回 `changedFiles: []`。
- `/Applications/Codex CN++.app/Contents/Info.plist` 可通过 `plutil -lint`。
- `/Applications/Codex CN++.app` 可通过 `codesign --verify --deep --strict`。
