# Codex CN++ v0.1.2

本版增强分发可靠性，重点是让其他用户下载后也能更稳定运行。

## 下载哪个文件

- `CodexCNPlusPlus-macos.zip`：推荐普通用户下载。解压后得到 `Codex CN++.app`，可拖入 `/Applications`。
- `CodexCNPlusPlus-macos-0.1.2.dmg`：DMG 安装包，适合习惯拖拽安装的用户。
- Source code：适合开发者继续维护词典、补丁规则和打包脚本。

## 本版修复

- 菜单栏 App 启动工具时优先使用内置 Node。
- 如果内置 Node 不可用，会继续查找 Homebrew、系统路径和 `/usr/bin/env node`。
- GUI App 启动时补充常见 PATH，避免别人双击 App 时找不到命令行环境里的 Node。
- 新增测试覆盖 macOS App 的 Node 兜底启动逻辑。

## 兼容说明

- 目标是安装了 Codex 桌面版的 macOS 用户。
- 汉化目标使用当前用户的 `$HOME/.codex` 和 `/Applications/Codex.app`。
- 如果 Codex 装在非标准位置，当前版本不会自动扫描自定义 App 路径。

## 本地验收

本版发布前确认：

- `npm test` 全部通过。
- `npm run dry-run` 在已汉化状态下返回 `changedFiles: []`。
- `/Applications/Codex CN++.app/Contents/Info.plist` 可通过 `plutil -lint`。
- `/Applications/Codex CN++.app` 可通过 `codesign --verify --deep --strict`。
